Помимо обычного оператора присвоения(=), существуют варианты совмещенные с математической операцией.
Операторы           Пример использования          Эквивалент
=                       x = y                       x = y
+=                      x += y                      x = x + y
-=                      x -= y                      x = x - y
_=                      x _= y                      x = x \* y
/=                      x /= y                      x = x / y
%=                      x %= y                      x = x % y
**=                     x **= y                     x = x \*\* y
